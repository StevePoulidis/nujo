## Installing and setting up

```bash
$ pip install nujo
```

Now run the following to make sure nujo was installed properly:
```bash
$ python
>>> import nujo as nj
>>> nj.__version__
'0.1.0'
```

***Read the docs at [vivelev.github.io/nujo](https://vivelev.github.io/nujo/)***
